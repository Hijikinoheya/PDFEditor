 概要
PDF Editorは、PDFファイルを簡単に編集できるスタンドアロンアプリケーションです。以下の機能を提供します。

- PDFの結合
- PDFの分割
- ページの削除
- PDFの回転
- PDFの暗号化・復号化
- 透かしの追加
- メタデータの編集
- PDFを画像に変換
- 画像をPDFに変換
- PDFをテキストに変換
- ページ順序の変更

 動作環境
- 動作可能機種: Windows x64
- 対応OS: Windows 10, Windows 11

 インストール方法
1. ダウンロードした`PDFEditor.exe`を任意の場所に保存します。
2. ダブルクリックしてアプリケーションを起動します。

 使用方法
1. PDFの結合:
   - 「PDFを結合」タブを選択し、結合したいPDFファイルを追加します。
   - 「PDFを結合」ボタンをクリックして、出力ファイルを保存します。

2. PDFの分割:
   - 「PDFを分割」タブを選択し、分割したいPDFファイルを選択します。
   - 出力フォルダを指定して、「PDFを分割」ボタンをクリックします。

3. ページの削除:
   - 「ページを削除」タブを選択し、PDFファイルを選択します。
   - 削除するページ番号を入力し、「ページを削除」ボタンをクリックします。

4. PDFの回転:
   - 「PDFを回転」タブを選択し、PDFファイルを選択します。
   - 回転角度を入力し、「PDFを回転」ボタンをクリックします。

5. PDFの暗号化・復号化:
   - 「PDFを暗号化」タブまたは「PDFを復号化」タブを選択し、PDFファイルを選択します。
   - パスワードを入力し、「暗号化」または「復号化」ボタンをクリックします。

6. 透かしの追加:
   - 「透かしを追加」タブを選択し、PDFファイルを選択します。
   - 透かしテキストを入力し、「透かしを追加」ボタンをクリックします。

7. メタデータの編集:
   - 「メタデータを編集」タブを選択し、PDFファイルを選択します。
   - タイトル、著者、サブジェクトを入力し、「メタデータを編集」ボタンをクリックします。

8. PDFを画像に変換:
   - 「PDFを画像に変換」タブを選択し、PDFファイルを選択します。
   - 出力フォルダを指定して、「PDFを画像に変換」ボタンをクリックします。

9. 画像をPDFに変換:
   - 「画像をPDFに変換」タブを選択し、画像ファイルを追加します。
   - 出力ファイルを指定して、「画像をPDFに変換」ボタンをクリックします。

10. PDFをテキストに変換:
    - 「PDFをテキストに変換」タブを選択し、PDFファイルを選択します。
    - テキストが表示されるので、必要に応じて保存します。

11. ページ順序の変更:
    - 「ページ順序を変更」タブを選択し、PDFファイルを選択します。
    - 新しいページ順序を入力し、「ページ順序を変更」ボタンをクリックします。

 注意事項
- 暗号化ファイルを削除する際は十分ご注意ください。削除されたファイルは復元できません。
- 安全なパスワードを使用してください。簡単なパスワードは攻撃に対して脆弱です。
